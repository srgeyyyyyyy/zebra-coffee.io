# Zebra Coffee ☕

Веб-сайт для кофейни Zebra Coffee в Астане с системой выбора филиалов и онлайн-заказом.

## 🚀 Функции

- **Выбор филиала** - Пользователи могут выбрать один из 10 филиалов Zebra Coffee в Астане
- **Онлайн-бронирование** - Удобная система бронирования столиков
- **Заказ со столика** - Возможность заказа еды и напитков прямо со столика
- **Адаптивный дизайн** - Полностью адаптивный интерфейс для всех устройств
- **3D анимации** - Красивые анимации чашки кофе с эффектами пара
